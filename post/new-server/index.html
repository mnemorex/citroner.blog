Ny server för webbsidan

New server for the website

Charlie Habolin
25 april 2020. 14:15

Nu har jag flyttat webbsidan från Github Pages till min egna server hemma, en Raspberry Pi 2 Model B. Anledningen till dett är att både få mer och enklare kontroll över websidan och webservern. Men också för att det är kul att kunna ha server hemma!

Severn drivs av Apache HTTP server som tillsammans med Let's Encrypt ger servern TLS certifikat.

Kontakta mig på charlie.habolin@monkcs.se om du har någon fråga!
